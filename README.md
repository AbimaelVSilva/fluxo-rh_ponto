# Apresentacao Ponto

Aplicacao estatica de apresentacao do sistema de ponto FluxoRH.

## Como executar

```bash
npm install
npm run dev
```

Abra a URL exibida pelo Vite no terminal.

Para gerar a versao de producao:

```bash
npm run build
```

## Estrutura

- `index.html`: pagina principal.
- `src/`: componentes JSX e estilos globais.
- `assets/`: imagens usadas pela aplicacao.
- `assets/uploads/`: imagens importadas/originais preservadas.
- `docs/archive/`: exportacao HTML original preservada.
- `docs/debug/`: capturas de depuracao preservadas.
